animation
屬性 說明
animation-name 動畫名稱
animation-duration 動畫持續時間，預設 0，單位 s 或 ms。
animation-delay 動畫延遲播放時間，預設 0，單位 s 或 ms。
animation-iteration-count

動畫播放次數，預設 1。
數字：播放次數。
infinite :動畫永遠重複播放。
animation-timing-function 動畫加速度函式，預設 ease。
其他還有： linear、ease-in、ease-out、ease-in-out、step-start、step-end、steps(int,start/end)、cubic-bezier(n,n,n,n)。
如果把上面的 step-start 改為 steps(3,start)，step-end 改為 steps(3,end)會變成抽格動畫。
animation-direction 動畫播放方向，預設 normal。
normal：動畫正常播放（向前）。這是默認值
reverse ：動畫反向播放（向後）
alternate ：動畫先播放，然後播放
alternate-reverse ：動畫先向後播放，然後向前播放
animation-fill-mode 動畫播放前後模式，預設 none。
none：默認值。動畫在執行之前或之後不會對元素應用任何樣式
forwards ：元素將保留由最後一個關鍵幀設置的樣式值（取決於動畫方向和動畫迭代次數），同最後一格freeze停止。
backwards ：元素將獲取由第一個關鍵幀設置的樣式值（取決於動畫方向），並在動畫延遲期間保留該值
both ：動畫將遵循向前和向後的規則，在兩個方向上擴展動畫屬性
animation-play-state 動畫播放或暫停狀態，預設 running。
running：預設值，表示動畫運行。
paused：表示動畫暫停。
寫法：animation:name duration | timing-function | delay | iteration-count | direction | fill-mode | play-state;

transition---------------
transition搭配a可以展示滑入滑出都很順暢，但transform滑出會有立即斷層。
CSS transitions 用來調整 CSS animation 變動的速度。
滑鼠請滑過，延遲.4s，很順暢
搭配animation
transition 使用shorthand屬性transition：
transition: width 2s linear 1s;		  
transition-delay 該transition-delay屬性指定過渡效果的延遲（以秒為單位）。  
transition-duration 用來定義 transition 屬性發生的時間，單位為秒。  
transition-property 用來定義可以產生 transition 屬性效果的屬性名稱，例如寬度、背景顏色 ...清單以逗號分隔。
all：所有屬性都將獲得過渡效果。
display、 position、 float & clear、box-model(margin、border、padding、wifth / height (寬高))、background、text、 font，https://andy6804tw.github.io/2017/12/19/css-common/		  
transition-timing-function
  • ease ：指定一個過渡效果，從慢速開始，然後快速，然後緩慢地結束（默認設置）
  • linear ：指定從開始到結束的速度相同的過渡效果
  • ease-in ：以慢速啟動指定過渡效果
  • ease-out ：指定過渡效果較慢的結尾
  • ease-in-out ：指定過渡效果，起點和終點較慢
  • cubic-bezier(n,n,n,n) ：讓您在三次方貝塞爾函數中定義自己的值
  
transform-origin 旋轉基礎點位置。
X 軸方向可以設定：left, center, right, 長度或百分比。
Y 軸方向可以設定：left, center, right, 長度或百分比。
